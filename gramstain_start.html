Stain sequence

Follow the four main steps of a Gram stain

The original Canvas file linked to missing branch pages. This version replaces that broken layout with a straight, readable walkthrough.

1

Apply crystal violet

This primary stain colors all cells purple at the start of the procedure.

Result: both Gram-positive and Gram-negative cells appear purple.
2

Add iodine

Iodine acts as a mordant. It helps the crystal violet form a larger complex that is harder to remove.

Result: the purple dye is more firmly associated with the cell wall.
3

Rinse with alcohol

This is the decisive step. Gram-positive cells retain the dye complex, while Gram-negative cells lose it.

Result: Gram-positive cells stay purple; Gram-negative cells become colorless.
4

Counterstain with safranin

Safranin stains the decolorized cells pink or red, making Gram-negative cells visible again.

Result: Gram-negative cells appear pink or red; Gram-positive cells remain purple.

Bottom line

If the cell ends purple, it is interpreted as Gram-positive. If it ends pink or red after the counterstain, it is interpreted as Gram-negative.
